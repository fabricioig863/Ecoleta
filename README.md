## 🚀 Projeto 

Aplicação desenvolvida durante o evento da primeira semana da Next Level Week da Rocktseat, em que a idéia seria ajudar pessoas a encontrar pontos de coleta de resíduos de uma forma mais conveniente.

## 💻 Linguagens:

- HTML
- CSS
- Javascript
- Node
- Express
- SQLite
